# Lambda-wav-sound-file-editor
This project is made with the intention to adress soundfiles through file handling in C. Though very limited, this serves the purpouse of managing segregated yet structured data.

## Assigned Task
```
The idea of ​​this activity is for you to manipulate a wave audio file. This file contains a header with basic information and then the uncompressed data.
You should create an interface so that the user can choose an audio file. Then the user can choose from several options:
1. Listen to the audio;
2. Extract a part of the audio, creating a new file, choosing a time interval, where the user chooses the start time and the end time;
3. Change the audio amplitude (volume);
4. Any additional function that you deem interesting;
```

## Features (functions)
1. Play audio;
2. Trim audio;
3. Change volume;
4. Change speed;
5. Change bit quantity per sample (de/compress);
6. Change channel quantity (mono/stereo);
7. Show soundfile data;
